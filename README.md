# SAS · Sumatif Akhir Semester
### Sekolah Rakyat Terintegrasi 64 Soppeng

Aplikasi ujian berbasis web (Sumatif Akhir Semester) untuk Sekolah Rakyat Terintegrasi 64 Soppeng. Dibangun dengan HTML, CSS, dan JavaScript murni — tidak membutuhkan server backend.

---

## 🚀 Akses Online

> **[Buka Aplikasi →](https://[USERNAME].github.io/[NAMA-REPO]/sumatif-akhir-semester.html)**

*(Ganti `[USERNAME]` dan `[NAMA-REPO]` sesuai akun dan nama repositori GitHub Anda)*

---

## 📋 Fitur

- **Mode Guru** — Buat & kelola soal (Pilihan Ganda, Pilihan Ganda Kompleks, Menjodohkan, Esai), atur kode ujian, unduh rekap nilai ke Excel
- **Mode Siswa** — Kerjakan ujian dengan timer, navigasi soal, submit otomatis
- **Rekap Nilai** — Export Excel otomatis dengan rumus nilai akhir, sheet detail jawaban, dan ringkasan

## ⚙️ Cara Deploy ke GitHub Pages

1. **Fork / Upload** repositori ini ke akun GitHub Anda
2. Masuk ke **Settings → Pages**
3. Pilih **Source:** `Deploy from a branch`
4. Pilih **Branch:** `main` → folder `/` (root) → klik **Save**
5. Tunggu ±1–2 menit, lalu akses di:
   ```
   https://[USERNAME].github.io/[NAMA-REPO]/sumatif-akhir-semester.html
   ```

---

## 💡 Catatan

- Data ujian disimpan di **localStorage** browser — tidak dikirim ke server mana pun.
- Untuk reset data, gunakan tombol **Hapus Data** di panel Guru.
- Pastikan siswa menggunakan browser modern (Chrome / Edge / Firefox terbaru).
